1.NumPy概述

NumPy(Numerical Python)是用Python进行科学计算的基础软件包。包含以下特点：

  1. 强大的N维数组对象Array
  2. 成熟的函数库
  3. 用于集成C/C++和Fortran代码的工具
  4. 实用的线性代数、傅立叶变换和随机生成函数
